# ImposterGame
Mobile and website Imposter Game

first check your IPv4 address using ipconfig in cd
To run mobile version, run a python http server through the folder with the games in its cd (Ex: python -m http.server 5500). 
Then on safari in mobile run http://{IPv4}:5500/ImposterGameMobile.html

example would be: http://192.168.1.20:5500/ImposterGameMobile.html

enjoy!
